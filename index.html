Бесплатный прокси за 3 минуты

Браузер не может обращаться к WB API напрямую (нет CORS). Cloudflare Worker бесплатен до 100 000 запросов/день.

  1. Откройте dash.cloudflare.com → Workers → Create → Worker.
  2. Удалите шаблон, вставьте код ниже, Deploy.
  3. Скопируйте URL вида https://wb-proxy.…workers.dev.
