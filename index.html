Бесплатный прокси за 3 минуты

Браузер не может обращаться к WB API напрямую (нет CORS). Cloudflare Worker решает это и бесплатен до 100 000 запросов в день.

  1. Откройте dash.cloudflare.com → Workers & Pages → Create → Worker.
  2. Удалите шаблон, вставьте код ниже, нажмите Deploy.
  3. Скопируйте адрес вида https://wb-proxy.…workers.dev в поле «URL прокси».
